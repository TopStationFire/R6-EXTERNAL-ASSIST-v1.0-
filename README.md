## ℹ️ What is R6 External Assist?
**R6 External Assist** is a standalone external overlay for Rainbow Six Siege. Unlike internal solutions that inject code, this tool operates entirely from outside the game process, providing visualization and assistance features with minimal detection risk. It renders a transparent DirectX 11 overlay that works in borderless windowed mode.

### ✨ Key Features
- **External Operation**: No code injection, no DLL loading. Pure memory reading via Win32 API.
- **ImGui Overlay**: Modern DirectX 11 transparent overlay with low performance impact.
- **Modular ESP**: Configurable boxes, skeletons, health bars, and name tags.
- **Aim Correction**: Smooth targeting system with adjustable FOV and smoothness.
- **Config System**: Save and load your preferred settings.
- **Auto-Update**: Offset database updated for each game patch.

## 🎛️ How to Use
1. **Download** the `R6_External_Assist.rar` from Releases.
2. **Extract** to any folder.
3. Launch Rainbow Six Siege in **Borderless Windowed** mode.
4. Run `R6_External_Assist.exe` as **Administrator**.
5. Press **INSERT** to toggle the control panel.
6. Configure features and enjoy.
